# GPS 可视化工具

GPS 轨迹可视化桌面应用

## 功能

- 📍 可视化 GPS 轨迹
- 🗄️ 支持 SQLite DB 文件
- 📁 支持 CSV 文件
- 🔍 自动识别经纬度字段
- 🔄 检查更新

## 下载

- [macOS 版本](https://github.com/YOUR_USERNAME/gps-visualizer/releases)
- [Windows 版本](https://github.com/YOUR_USERNAME/gps-visualizer/releases)

## 使用

1. 下载对应系统的安装包
2. 安装并打开应用
3. 点击"打开 DB"或"打开 CSV"导入数据
4. 在浏览器中查看轨迹
